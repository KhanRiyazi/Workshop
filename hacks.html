1. Build Small APIs Daily

Practice micro-APIs to build muscle memory.

Mini Challenges

Goal: Create endpoints for add, subtract, multiply, divide.
# FastAPI sample (calculator)
from fastapi import FastAPI
app = FastAPI()

@app.get("/add")
def add(a: float, b: float):
    return {"result": a + b}
Goal: Use query params like ?city=Delhi and return mock weather.
# Example: /weather?city=Delhi
@app.get("/weather")
def weather(city: str = "Delhi"):
    return {"city": city, "temp_c": 29.5, "condition": "Sunny"}
